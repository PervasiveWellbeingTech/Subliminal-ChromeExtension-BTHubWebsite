Breathing Edges
Adaptive Mode

Adaptive mode enables pairing an external bluetooth sensor to Breathing Edges to get breathing and heart rate data. The extension will then turn on at the appropriate rate for the user only when needed

1. Enable bluetooth on your computer and make sure external device is paired

2. Click the start button to connect extension to sensor

3. Browse normally without closing this tab. You can switch to other tabs as usual
